
Instructions to use the bimanual planner-
Locate the bimanual planner file in the ManiSkill directory of the branch.
Verify that the version of mplib you are using is 0.1.1
Copy it to the location of mplib, in the same directory planner.py is present.
Run the mani_skill/examples/motionplanning/dual_panda/solutions/bimanual_test.py for a quick demo
